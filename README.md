# score_timer

## brief
A little script for DCS that will show the score for Red and Blue team at the bottom left screen. No more spamming the history!
<img src=doc/Ad.jpg>

## installation
add the score_timer.lua into the mission start trigger after you initialized a "BlueScore" and a "RedScore" flag.
<img src=doc/MissionEditor.jpg>


Save the file, exit the MissionEditor and open the .miz file to the drag & drop the score_timer folder with the pictures into the root folder of the mission.
Use something like 7zip for it.
<img src=doc/DropFiles.png>
